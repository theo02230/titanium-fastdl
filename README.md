# Titanium Basewars FastDL

GMod Fast Download mirror for Titanium Basewars server.

- `sv_downloadurl` target — assets compressed in `.bz2` (GMod tries `.bz2` first, falls back to original).
- Auto-generated from local `addons/` (excluding addons available on Steam Workshop).
- Structure mirrors `garrysmod/` root: `materials/`, `models/`, `sound/`, `particles/`, `resource/`.
